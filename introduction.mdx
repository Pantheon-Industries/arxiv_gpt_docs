## Overview

Welcome to the arXiv Machine-Readable Paper API, a powerful tool for retrieving machine-readable versions of arXiv papers in text or markdown format.

Our API provides access to a vast collection of academic papers from arXiv, allowing you to easily integrate scholarly content into your applications, research tools, and analysis pipelines.

## Getting Started

To use the API, you'll need an API key. Once you have your key, you can start making requests to our endpoints.

### Authentication

Include your API key in the header of each request:

X-API-Key: YOUR_API_KEY_HERE

## Quick Start Guide

Here's a quick example of how to use our API:

```python
import requests

api_key = "YOUR_API_KEY_HERE"
paper_id = "2408.01800"
max_pages = 5

headers = {
    "X-API-Key": api_key
}

url = f"https://boiling-savannah-93394-fe89b9dd363e.herokuapp.com/papers/text?paper_id={paper_id}&max_pages={max_pages}"

response = requests.get(url, headers=headers)

if response.status_code == 200:
    data = response.json()
    print(f"Paper ID: {data['paper_id']}")
    print(f"Content: {data['content'][:500]}...")  # Print first 500 characters
    print(f"Total Pages: {data['total_pages']}")
    print(f"Returned Pages: {data['returned_pages']}")
    print(f"Format: {data['format']}")
else:
    print(f"Error: {response.status_code}")
    print(response.json())

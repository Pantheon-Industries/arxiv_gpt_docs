# Get Paper as Text

Get a plain text version of an arXiv paper.

## Endpoint

GET /papers/text

## Authorization

| Name | Type | In | Description |
|------|------|----| ------------|
| X-API-Key | string | header | required |

API key for authentication.

## Query Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| paper_id | string | required | The arXiv ID of the paper |
| max_pages | integer | optional | Maximum number of pages to return (default: all) |

## Response

**200 - application/json**

```json
{
  "paper_id": "string",
  "content": "string",
  "total_pages": 0,
  "returned_pages": 0,
  "format": "text"
}

## Example

import requests

headers = {
    "X-API-Key": "your_api_key_here"
}

params = {
    "paper_id": "2408.01800",
    "max_pages": 5
}

url = "https://boiling-savannah-93394-fe89b9dd363e.herokuapp.com/papers/text"

response = requests.get(url, headers=headers, params=params)

if response.status_code == 200:
    data = response.json()
    print(f"Paper ID: {data['paper_id']}")
    print(f"Content: {data['content'][:500]}...")  # Print first 500 characters
    print(f"Total Pages: {data['total_pages']}")
    print(f"Returned Pages: {data['returned_pages']}")
    print(f"Format: {data['format']}")
else:
    print(f"Error: {response.status_code}")
    print(response.json())
